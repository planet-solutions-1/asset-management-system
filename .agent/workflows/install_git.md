---
description: How to install Git on Windows
---

1.  **Download**: Go to [git-scm.com](https://git-scm.com/download/win) and download the latest "64-bit Git for Windows Setup".
2.  **Install**: Run the downloaded `.exe` file.
3.  **Setup**: Click "Next" through the installation screens.
    *   **Important**: When you reach the screen "Adjusting your PATH environment", ensure **"Git from the command line and also from 3rd-party software"** is selected.
4.  **Finish**: Click "Install" and wait for it to finish.
5.  **Restart**: **You must restart your computer** (or at least completely close and reopen VS Code) for the changes to take effect.
